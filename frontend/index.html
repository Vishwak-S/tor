System Overview

TOR-Unveil correlates network traffic with TOR topology to identify probable entry guard nodes. The system uses temporal alignment, bandwidth feasibility analysis, and flow fingerprinting to generate forensic-grade evidence trails.

Workflow:

  1. Crawl Topology: Fetch current TOR consensus and relay descriptors
  2. Upload PCAP: Ingest captured network traffic for analysis
  3. Run Correlation: Match flows to probable entry guards
  4. Generate Report: Export findings in PDF/CSV format
